# gym-db
Repositório do trabalho prático da disciplina de Banco de Dados, com foco na modelagem e implementação de um sistema para gerenciamento de academia.

## 👥 Integrantes

## 📊 Objetivo
Desenvolver um banco de dados para gerenciar alunos, planos, treinos e instrutores de uma academia.

## 🛠 Tecnologias
- SQL (MySQL)

## 📁 Estrutura do Projeto
- scripts/ddl → criação das tabelas
- scripts/dml → inserção de dados
- scripts/queries → consultas
